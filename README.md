# HOSxP Data Pipeline & Analytics

โปรเจกต์วิเคราะห์ข้อมูลคนไข้ขนาดใหญ่ (Big Data) จากระบบ HOSxP โดยใช้ Python และเทคโนโลยีสมัยใหม่ เพื่อช่วยในการตัดสินใจทางการแพทย์และการบริหารจัดการโรงพยาบาล

## Features
- **High-Performance Processing:** รองรับการวิเคราะห์ข้อมูลระดับ 10-100 ล้านแถวด้วย PySpark
- **Real-time Dashboard:** แสดงผลกลุ่มเสี่ยง (Risk Stratification) ผ่าน Streamlit
- **Professional UI:** รายงานสรุปผลในรูปแบบตารางและกราฟที่โต้ตอบได้ (Interactive)

## Tech Stack
- **Language:** Python 3.9+
- **Data Engineering:** PySpark, Pandas, NumPy
- **Visualization:** Streamlit, Plotly
- **Terminal UI:** Rich
